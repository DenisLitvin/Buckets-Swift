Deque

A double-ended queue (deque) is a collection that generalizes a queue, for which elements can be added to or removed from either the front or back.

The enqueueFirst, enqueueLast, dequeueFirst and dequeueLast operations run in amortized constant time. Comforms to SequenceType, ArrayLiteralConvertible, Printable and DebugPrintable.
