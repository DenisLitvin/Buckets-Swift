Stack

A Stack is a Last-In-First-Out (LIFO) collection, the last element added to the stack will be the first one to be removed.

The push and pop operations run in amortized constant time. Comforms to SequenceType, ArrayLiteralConvertible, Printable and DebugPrintable.
