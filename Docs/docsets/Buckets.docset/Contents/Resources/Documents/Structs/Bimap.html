Bimap

A Bimap is a special kind of dictionary that allows bidirectional lookup between key and values. All keys and values must be unique. It allows to get, set, or delete a key-value pairs using subscript notation: bimap[value: theValue] or bimap[key: thekey]

Comforms to SequenceType, CollectionType, DictionaryLiteralConvertible, Equatable, Hashable, Printable and DebugPrintable.
